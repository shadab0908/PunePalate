# Kya-Khaega 🍽️

> Confused about what to eat? Let Kya-Khaega decide for you!

Kya-Khaega is a simple full-stack food suggestion web application built as part of the **PunePalate** project. It helps users discover food options quickly through a clean interface connected to a backend API.

This project demonstrates frontend-backend integration and basic full-stack architecture.

---

## 🚀 Features

- 🎯 Random food suggestions
- 🖥️ Interactive frontend UI
- 🔌 Backend API integration
- ⚡ Fast local setup
- 📚 Beginner-friendly full-stack structure

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- HTML
- CSS
- JavaScript
- npm

**Backend**
- Python
- Flask (or similar lightweight framework)

**Architecture**
- REST API based communication
- Separate frontend and backend folders

## 💻 How to Run Locally

### 1️⃣ Start the Frontend

Open terminal:

```bash
cd frontend
npm install
npm start
```

Frontend will run on:
```
http://localhost:3000
```

---

### 2️⃣ Start the Backend

Open another terminal:

```bash
cd backend
python app.py
```

Backend server will start on:
```
http://localhost:5000
```

Make sure both frontend and backend are running simultaneously.
